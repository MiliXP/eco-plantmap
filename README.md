# 🌱 PlantMap

## 📌 Sobre o Projeto

O **PlantMap** é um sistema web desenvolvido para incentivar a sustentabilidade urbana, permitindo o registro de árvores plantadas e a visualização de pontos ecológicos em um mapa interativo.

O objetivo é facilitar o monitoramento ambiental e promover ações sustentáveis, como reciclagem e uso de energia limpa.

---------------------

## 🚀 Funcionalidades

* 🌳 Cadastro de árvores com localização geográfica.
* 📍 Visualização de árvores no mapa.
* ♻️ Pontos de coleta seletiva.
* 🔋 Pontos de coleta de lixo eletrônico.
* ⚡ Estações de recarga para carros elétricos.
* 📷 Upload de fotos das árvores.
